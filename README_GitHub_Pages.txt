# Despliegue rápido en GitHub Pages

## 1) Crear repositorio
- Nombre sugerido: `hernia-hc` (o el que prefieras). Público.

## 2) Subir estos archivos (todos en la raíz)
- `index.html`
- `HistoriaClinica_HerniaVentral_Ambulatorio_v1_7_3.html`
- `HistoriaClinica_HerniaParaostomal_v1_1.html`

## 3) Activar GitHub Pages
- Settings → **Pages**
- Source: **Deploy from a branch**
- Branch: `main` / carpeta **root**
- Guardar.

## 4) Probar
- Abre la URL: `https://TU-USUARIO.github.io/hernia-hc/`.
- Si el portapapeles da error, usa Chrome/Edge y https (GitHub Pages lo da por defecto).

## Notas
- El favicon y el logo están incrustados como SVG para que no tengas que subir imágenes aparte.
- Puedes editar texto/estilos directamente en cada `.html`.
